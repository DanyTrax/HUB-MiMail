# Alcance del Proyecto

## Incluye
- Migracion de correos entre plataformas soportadas.
- Backups periodicos por lista de cuentas.
- Multiempresa con aislamiento de datos.
- Multiusuario con roles y permisos.
- Programacion de tareas (cron) y ejecuciones manuales.
- Logs tecnicos y auditoria de acciones.

## No incluye
- Visualizacion de correos por webmail dentro de esta plataforma.
- Provisionamiento de Roundcube/SnappyMail (se maneja por contenedor externo).
